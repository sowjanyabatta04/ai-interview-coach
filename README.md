# AI Interview Coach

AI Interview Coach is a Streamlit-based web application that simulates real interview experiences.

## Features
- Random interview questions
- Resume upload
- Resume-based personalized questions
- 60-second timer per question
- AI-based answer evaluation
- Score and feedback
- Downloadable PDF report

## Technologies Used
- Python
- Streamlit
- ReportLab
- NLP / AI
